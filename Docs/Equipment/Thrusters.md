# Thrusters

There are many different kinds of thrusters available that all provide differing outputs of energy, agility, and speed. All thrusters are the same. The category is essentially an assembly of single thrusters. A thruster can have a range of properties based on quality and purpose; a balance of thrust output and efficiency (the higher the output, the worse the efficiency and power consumption).

| Type                           | Description                                                                                                                                                                                        |
| ------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Unary Thruster                 | A single thruster which can have a set range of properties. Small ships only have space for a single primary thruster and cannot house more than the one.                                          |
| Binary Thruster                | A set of two Unary Thrusters that work in sync to produce 2x thrust of their unary counterpart.                                                                                                    |
| Ternary Thruster               | A set of three Unary Thrusters that work in sync to produce 3x the thrust of their unary counterpart.                                                                                              |
| Omnidirectional Thruster Array | An array small Unary Thrusters that are used to increase omnidirectional agility and maneuverability. These do not provide much propulsion, but can make a ship turn, bank, and shift much faster. |

Ships will have a property stating the maximum size of thruster it can physically hold. This limits the amount of speed a small ship can have. Also, smaller ships can only house smaller power plants and are unable to power certain high-output engines.



## Properties

Every thruster has three properties, Maximum Velocity, Acceleration, Power Consumption.

| Property          | Description                                                                                      |
| ----------------- | ------------------------------------------------------------------------------------------------ |
| Maximum Velocity  | The maximum velocity the engine can achieve, limited by the speed of propulsion matter ejection. |
| Acceleration      | The amount of Force an engine can provide. This is directly affected by the ship's Mass.         |
| Power Consumption | How much power is needed to operate the engine.                                                  |
